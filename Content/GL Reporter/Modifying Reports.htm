3 Modifying Reports

3.1 Change Report Rows

Existing report rows may be changed by amending, deleting or copying rows. 

3.1.1 Amend Report Rows

To amend a Report Row select the line to be amended by using the yellow Open button. The line is then available for amendment.

 

3.1.2 Delete Report Rows

    To delete a Report Row or multiple rows, select the line(s) to be deleted by putting a in the box before the line number(s).

    Select the Dustbin button at the bottom of the browse screen to delete the line. 

    A message will warn that deleting the line will be permanent and select OK to continue and delete the line. If multiple lines are selected then the number of rows to be deleted will be shown in the message.

    The line(s) will be removed from the Report Rows.

 

3.1.3 Copy Report Rows

    Select the line to be copied by ticking the box of the required line.

    Select the Copy button at the bottom of the browse screen

    The line selected will be copied and the line may be modified.

 

3.1.4 Move Report Rows

    Select the line(s) to be moved by ticking the box of the required line(s). It is possible to move a group of lines either by clicking individual tick boxes for lines or using ‘Shift+Click’ for a range of lines or ‘Ctrl+Click’ to add lines to already selected lines.

    Select the Move Up or Move Down buttons at the bottom of the browse screen.

    The selected lines will either be moved up or down one row at a time.
