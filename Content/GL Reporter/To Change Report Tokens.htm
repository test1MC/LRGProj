3.3 To Change Report Tokens

Report Tokens may be changed by amending, deleting or copying columns.

3.3.1 Amend Report Tokens

To amend a Report Token select the line to be amended using the yellow Open button. The line is then available for amendment.

3.3.2 Delete Report Tokens

    To delete a Report Token select the line to be amended by putting a in the box before the line number. 

    Select the Dustbin button at the bottom of the browse screen to delete the line. 

    A message will warn that deleting the line will be permanent and select OK to continue and delete the line.

    The line will be removed from the Report Tokens.

3.3.3 Copy Report Tokens

    Select the line to be copied by ticking the box of the required line.

    Select the Copy button at the bottom of the browse screen

    The line selected will be copied and the line may be modified.
