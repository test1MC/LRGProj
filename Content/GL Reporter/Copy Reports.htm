3.4 Copy Reports

General Ledger Reporter reports may be copied or cloned as the basis for creating a new report, which can then be modified as required. This copy function will copy all details of the report to a new report code within the same company. You must use the export report and import report functions to copy a report to a different General Ledger company.

    Select the report to be copied by ticking the box alongside the report.

    Select the Copy button

    The report will be copied and the Configuration tab will displayed for the new report.

    The new report can be given a new name and title.

    Save the Configuration details and move forward to the next screen.

    Make any changes required to the Report Row; Report Columns and if applicable the Report Tokens.
