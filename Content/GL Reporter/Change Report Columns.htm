3.2 Change Report Columns

Existing report columns may be changed by amending, deleting or copying columns.

3.2.1 Amend Report Columns

To amend a Report Column select the line to be amended by using the yellow Open button. The line is then available for amendment.

 

3.2.2 Delete Report Columns

    To delete a Report Column select the line to be amended by putting a in the box before the line number. 

    Select the Dustbin button at the bottom of the browse screen to delete the line. 

    A message will warn that deleting the line will be permanent and select OK to continue and delete the line.

    The line will be removed the Report Columns.

 

3.2.3 Copy Report Columns

    Select the line to be copied by ticking the box of the required line.

    Select the Copy button at the bottom of the browse screen

    The line selected will be copied and the line may be modified.
