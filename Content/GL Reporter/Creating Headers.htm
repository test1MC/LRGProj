1.5 Creating Headers

You may require using certain data in the report to show detail that is not available from fields or records within the system.

These could be Column Headings and therefore we are adding text and also placing it in the Header.

These are Literal columns, as they will not be available for the Row.

    Select Literal option from the Item field drop down list. Select the Insert button.

It is recommended that you keep the headers together and the body of the data together.

    Enter the Text that you wish to display in the Label field. Take care to type this correctly, as what you type will be displayed in the report.

    Enter the Row as a negative figure, as this will denote this is a header field and therefore not to be included in the body of the report.

As this is a Header you may also want to have a different font, or make the text Bold.

    Select the Save button.

You can run the report and the data will be displayed in the header of the report.
