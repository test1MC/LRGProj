Document Management Global Search

There is a new Document Management Global Search screen.

Global Search Screen

You must select one or more document categories to search, and you can search for text in any of the document fields that are configured for the search. You can also restrict the search to documents added between selected dates and/or added by selected people.

When you click Next, the results are shown. There are two views of the results: standard view (which is a standard browse) and card view. A filter selector allows you to switch between views.

Standard View

Standard view shows one line per document.

Search Results (Standard View)

Click the link in the Description column to view the document.

Filtering

The Keys column field lists all the keys associated with the document, separated with a /.

Actions

The following actions are available:

  • Email Documents
  • Check Out Document
  • Check In Document
  • Print Document
  • Copy Document
  • Export Documents

With the exception of Check In Document, the above actions can be run for one or more documents.

Card View

Card view gives a thumbnail view of the document if one is accessible.

Search Results (Card View)

Click the thumbnail to view the document.

Refine Search

The Refine Search tab allows you to change the search criteria and run the search again.
