Action Monitor

The Action Monitor is a version of the Activity Workbench Tasks tab, tailored to actions arising from H&S audits, and filtered to just these actions. There are tabs for the main sort/search you want to use.

Action Monitor

Actions always belong to a user, but can be assigned to other people (for example, to subcontractors or contacts).

Task Notes

On the task summary, there is a new tab for Notes on the task.  When a task is added the date, time and user are stamped on the record.  The notes browse has columns for date/time, user and the note text.  In display mode the text shows the first few lines.  Detail view shows the whole text of the note.

Task Email

An Email button has been added to the Action Monitor browse and the task summary.  When clicked this opens a page to allow you to enter details of the email. There are fields for Assignee type and Assignee, Email To and Cc, Subject and Message.  The default text for the subject and message are defined by the parameters TSKMSUBJ and TSKMBODY (see 1 , Health and Safety Parameters).

Task Email

When the email is sent, a task note is automatically created with details of the email and content sent, and if an Assignee was set the action is assigned to the person emailed.

If a task is already assigned, the Assignee and Email To will default from this. 

The Attachment tab lists any documents linked to the section and element of the form that the task relates to, with a box selection of whether the image should be attached. You can hover over the attachment to preview the image.
