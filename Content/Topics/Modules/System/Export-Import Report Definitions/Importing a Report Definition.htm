Importing a Report Definition

From the Options menu on the Report Builder List screen, choose the Import Report Builder option.

Paste in the report data you copied from the Export Report Builder action.

If any of the import parts already exist in the database then you will be asked to confirm that you want to overwrite them.

If you confirm this, the report section, form, fields, calculations, data source, dataset and fields, and collection dataset will be imported as required.

You can check that the data source, dataset and collection dataset have been created by looking in the corresponding maintenance pages for these items.

You will be able to load and run the newly imported report(s).
