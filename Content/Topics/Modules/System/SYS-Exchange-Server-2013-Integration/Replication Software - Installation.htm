Installing the Integration Application

The OA Outlook Integration Application needs to be installed on a Windows machine. It is not recommended to install it on the same server as is used for Exchange.

will provide the install files.

  1. Run setup.exe.

  2. Choose how you want to install the integration software.

    There are two ways for the integration to run.

    • Console: The program runs constantly in a console window, and polls the Exchange server at a specified interval. It can also be run as a scheduled Windows task. This is the preferred way for initial set up, as it gives better feedback.
    • Windows Service: This integrates the Exchange programs directly via the windows services.

    You can choose to install the integration software to run as a Windows service, or a console program, or both.
