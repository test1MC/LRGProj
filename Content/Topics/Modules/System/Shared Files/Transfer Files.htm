Transfer Files

There is a new An option on the My Files tab which allows you to transfer files to other shared folders (assuming you have access to them).

My Files - Transfer Files

The selected files are shown, together with an option to copy or to move the file to another one of the available folders (this checks that you have WRITE permission).

Transfer Files Selection Screen

Selecting the Next button will either Copy or Move the file(s).

You will not be allowed to overwrite a file that already exists in the target directory.
