Copy Reports

General Ledger Reporter reports may be copied or cloned as the basis for creating a new report, which can then be modified as required. This copy function will copy all details of the report to a new report code within the same company. You must use the export report and import report functions to copy a report to a different General Ledger company.

  1. Select the report to be copied.

  2. Click the Copy button
  3. The report will be copied and the Configuration tab will displayed for the new report.

  4. Give the new report a new name and title.
  5. Save the Configuration details and move forward to the next screen.
  6. Make any changes required to the Report Row; Report Columns and if applicable the Report Tokens.
