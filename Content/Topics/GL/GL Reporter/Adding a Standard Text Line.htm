Adding a Standard Text Line

Standard Text Lines in report rows enable a user to enter their own free text to enhance a report. This would also include underlining.

  1. Select Standard Text Line as the row type option from the field dropdown list, and click the Add button.

    Change screen

    A new line is created for a Standard Text Line in the report.

  2. Enter the description for the Standard Text Line field to be printed on the report e.g. Trading Accounts

    Change Screen

    The Detail and Print fields will be selected automatically in order to print the Standard TextLine on the report.

  3. Click the Save button.

    A new line is displayed. Click the Undo button.
