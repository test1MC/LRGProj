Adding an Additional Text Line

An Additional Text Line in report rows enables a user to enter more than one line of free text on a report, for example an address or a paragraph.

  1. Click the Additional Text Line option from the type field column drop down list and click the Add button.

    A new line is created for an Additional Text Line in the report.

    SCREEN SHOT

  2. Enter the description for the Additional Text Line to be printed on the report e.g. EMEA

    The Detail and Print fields will be automatically default in order to print the Additional TextLine on the report.

    3. A large text field is then available in order to enter the additional text to be printed on the report e.g. EMEA DIVISION = Europe, Middle East and Africa.

  3. Click the Save button.

    A new line is displayed. Click the Undo button.
