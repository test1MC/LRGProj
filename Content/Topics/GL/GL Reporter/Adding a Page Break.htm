Adding a Page Break

Add a Page Break to start a new page before continuing the display of balances.

  1. Select Page Break as the row type option from the type field drop down menu list, and click the Add button.

    NEW SCREENSHOT

    A new line is created for the Page Break in the report.

  2. Enter the description for the Page Break Line e.g. Page Break

    3. The Detail and Print fields will automatically default in order to print the Page Break on the report.

  3. Click the Save button.

    A new line is displayed select the Undo button.
