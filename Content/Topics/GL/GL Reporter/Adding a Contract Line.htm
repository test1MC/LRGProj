Adding a Contract Line

A Contract Line report row will display balances on the report, based on the Contact selections entered in the line.

  1. Select Contract Line as the row type option from the type field drop down list and click the Add button.

    A new line is created for the Contract Line in the report.

  2. Enter the description for the Contract Line to be printed on the report e.g. Africa Contracts

    The Detail and Print fields will automatically default in order to print the Contract Line on the report.

  3. Enter the Contract Line detail in the relevant fields. The fields are defined in the table below.

    Field:

    Description:

    Level:

    Contract lines at the same level will be displayed together in a column on the report. 

    Contract:

    The Contract number or range of contracts for this line. 

    Group:

    The Group to report on. 

    Type:

    The Type or range of Types to report on.

    Location:

    The Location or range of Locations to report on.

    Manager:

    The Manager or range of Managers to report on.

    Client:

    The Client or range of Clients to report on.

    Status of Contract:

    The Status of Contracts or a range to report on.

    Active:

    Use the drop down menu to select if All, xxx.xxxx should be retrieved for the report.

    Complete:

    Use the drop down menu to select completed, xxxxx, or xxxx Contracts.

    Include Future Contracts:

    To exclude future contracts in the report balances.

    Company Group:

    The Company Group to report on. 

    Currency Type:

    The base currency for conversion if you are using more than one currency.

    Percent Line:

    The line number to reflect as a % of another row e.g. Profit as a % of Total Sales. 

    Divide Row:

    The line number to divide a row by another row. 

    Sort:

    The order to sort the report.  Max 5 levels e.g. 1 = Account, 2= Analysis Code, 3 = Branch, 4 = Dept, 5 = Detail. 

    Header/Footer Level:

    The level corresponds to a set of header and footer column fields configured in Report Columns.

    Print Line:

    Print the line on the report or just process to accumulate report values.

    Print Zero:

    Print zero balances on the report.

    Reverse Value:

    Display negative values as positive and vice versa.

  4. Click the Save button.

    A new line is displayed select the Undo button.

When all the report rows have been entered, navigate to the Report Columns Tab to continue configuring the report.
