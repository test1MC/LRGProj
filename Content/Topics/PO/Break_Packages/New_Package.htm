Placing Orders for a Package with Different Suppliers

When comparing supplier quotations on the Comparison tab, you can choose to place more than one order by moving some of the items to a new, separate package.

To move items to a new package

  1. Select the item you want to add to a different order, or use Shift+click or Ctrl+click to select more than one item.
  2. Right-click a selected item then choose Move Items to New Package.

    The Move Items to New Package screen is displayed. This lists the selected items.

    Move Items to New Package Screen

  3. Optionally, enter a code and description for the new package. You can use the same code and description if you want to show that it relates to the original package.
  4. Click Next.

    A new package is created and inserted into the list of packages immediately below the originating package.

    Packages Tab – New Package

    The letter “S” (for Split) is shown to the right of the description of the new package. A tooltip on this value shows details of the originating package (Seq, Package Code and Description). The Tender Allowance on both the originating package and the newly-created package are recalculated to reflect the split. The details of any quoted prices for the items are included in the new package.
