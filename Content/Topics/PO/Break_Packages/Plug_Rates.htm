Price - Plug Rates

If you receive quotations from suppliers that do not include rates for all requested items, you may want to enter representative rates to allow you to make a realistic comparison between different quotations. You can do this by entering ‘plug rates’.

To enter plug rates for a supplier

  2. Go to the Comparison tab.
  3. Click the supplier name.

  4. This displays a frame with the items from the schedule.

  5. Do one of the following:
    • Update a row to enter the plug rate manually. You can enter comments to explain why you have updated the rate.
    • Run the Plug Lowest action to enter the lowest quote for each item.
    • Run the Plug Average action to enter the average of the quotes for each item.
    • Run the Plug Highest action to enter the highest quote for each item.
  6. Close the Enter Plug Rates and Comments window and click to refresh the matrix display.
