Land Appraisal Main Workbench

 

To access the Main Workbench, navigate to Land Appraisal > Main Workbench

 

Land Appraisals By Development Tab

By Development shows a list of developments and a summary of the number of appraisals for that development. You can dig deeper to see a full list of the appraisals.

 

Land Appraisals My Appraisals Tab

My Appraisals shows all of the appraisals for sites on which you have appraisals checked out. From here, you can check out new appraisals, check existing appraisals back in, or cancel them.

This allows for a small set of relevant records to be displayed on one screen so that you can more easily keep track of the records they you currently working with.

 

Land Appraisals By Checked Out Appraisal Tab

By Checked Out Appraisal displays a list of the appraisals that are being actively monitored by users in the system. The appraisals can be filtered by an individual user, development or appraisal.

 

Land Appraisals By Appraisal Tab

By Appraisal shows a full list of all the appraisals within Land Appraisal and a summary of the key information for each one.

You can dig deeper to see the full appraisal, although it cannot be updated unless checked out first.

 

Land Appraisals By Bid Development Tab

By Bid Development shows a list of bid developments and a summary of the number of bids for that development. Bids are summary versions of appraisals that can be entered more quickly for quick decisions on land viability. If a bid goes ahead it can be converted into an appraisal.

You can dig deeper to see a full list of the bids.
