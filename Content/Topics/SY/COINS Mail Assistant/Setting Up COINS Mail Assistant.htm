Setting Up Mail Assistant

Setting up Mail Assistant involves the following tasks:

  1. Install the Mail Assistant manifest file in Exchange or Office 365 (see 1, Modifying and Installing the Manifest File).
  2. Install OA Outlook Integration Application (see Installing OA Outlook Integration Application).
  3. Set up users to use Mail Assistant (see COINS Mail Assistant - OA Configuration).

NOTES

The needs to include the add-in role with the designated number of users.

HTTPS should be enabled with a valid SSL Certificate on the web server for the OA environment.
