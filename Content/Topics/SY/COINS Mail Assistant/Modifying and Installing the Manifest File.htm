Modifying and Installing the Manifest File

The Mail Assistant manifest is an XML file that tells the Outlook client where to find the Mail Assistant and its associated files. The manifest file is provided as part of the Mail Assistant installation package.

To use Mail Assistant, each environment needs a different version of the manifest file. Unless has provided you with a pre-configured version of the file, you will need to modify the file to point to the correct locations on your environment (see 1, Modifying the COINS Mail Assistant Manifest File). Then the manifest file needs to be installed on Exchange and made available to your users (see 1, Installing the COINS Mail Assistant Manifest File).
