Updating Triggers for

Triggers allow you to keep track of when events related to or vehicle assets are due. You can automatically update hourly and daily triggers for which is .

For example, suppose you have a item which you want to inspect when it has been for 60 days or more. Using Asset Maintenance.

Trigger Description TT Value Expires Last Reset Last Upd
INSP-60D 60 Days Insp GE 0.0 60.0 10/06/11 10/06/11

You should update the daily (and hourly) triggers periodically, and before reporting on trigger dates, for example. This will update the value of the trigger; when this value reaches (or exceeds) the Expires value (60 in this example), the trigger will fire, and will appear on the Asset Triggers Report.

To update triggers:

  1. Go to Update On Triggers to update the same trigger.  You could have different triggers that use the different methods; so you could use TRIGHOUR to drive hourly-based servicing, and have a trigger for inspection status updated by a caption.
