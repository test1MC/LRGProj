Exporting and Importing Function Security

You can export the function security settings for a user or group from one system and import them to another.

To export function security:

  1. Run Export Function Security.
  2. Specify the user or group whose security settings you want to export.
  3. Enter a can-do list of the functions whose settings you want to export.
  4. If you want to export the full list of functions (not just those with explicit Yes or No security settings), the Export Group Access? box. 
  5. Click .
  6. outputs the settings to a Function Security Data field.
  7. Copy the contents of the field.
    1. Right-click the field.
    2. Choose Select All.
    3. Right-click again.
    4. Choose Copy
  8. Paste the contents into a file (or import directly into another system using Import Function Security). 

To import function security:

  1. Run Import Function Security.
  2. Paste the function security data (exported using Export Function Security) into the Function Security Data field.
