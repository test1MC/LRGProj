Sending Notifications Using the Notifications Browse

Mail merge is available from the Notifications browse, though the Notifications Report (see Sending Notifications Using the Notifications Report) is expected to be used to produce the majority of notifications.

If you use the mail merge to produce notifications, you will need to fill in the sent dates manually.

To produce notifications using the Notifications Workbench:

  1. Use the filters to get to the list of notifications you want to send.  This will be a single notification type, typically only unsent notifications, and you might want to filter to those with or without an email address.
  2. Select the employees for whom you want to produce the notifications, or leave all rows unselected to merge all employees in the given filter criteria.
  3. Click the Merge button  .
  4. Choose the merge file you want to use, then click the action button  .
