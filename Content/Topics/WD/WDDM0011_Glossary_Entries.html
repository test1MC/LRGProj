Glossary Entries

Glossary entries show a brief explanation of a word or expression when the user clicks them.  Use glossary entries to explain technical terms, or words or phrases that have a special meaning in .

To insert a glossary entry:

  • In the text, type glossary_token.  For example: $G$candolst$G$.  (Set up glossary entries using Glossary.)
  • To use different text for the glossary link (for example, to display a plural, where the glossary item is singular), type alternative text.  For example: $G$planorde| orders$G$.
  • You can also use the lookup: select Glossary.
