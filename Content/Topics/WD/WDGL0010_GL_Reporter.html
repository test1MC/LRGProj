GL Reporter

The General Ledger Reporter allows you to produce reports based on General Ledger accounts; for example:

  • Profit and loss reports
  • Balance sheet reports
  • Overhead analysis
  • Budget/actual comparisons

You can also report on some information, such as House Sales budgets and report stores.

You can report on GL accounts from all sectors of the current , but also from other in the same environment.  The reports can be detailed or summared, and the accounts that are included can either be specified as part of the report configuration, or chosen by the user when they run the report.

To create a GL report:

  1. Go to Reporter Configuration.
  2. Give the report a code and a title. 

    These are used to identify the report, for example when choosing from a list which report to run.  The code is the only part of the report that you cannot change directly once you have chosen it. If you need to rename a report you must copy it.  The title will be printed on the report.

  3. On the Configuration tab, enter the settings that determine the overall appearance of the report, such as the page layout and font class to use.  See the side-frame help for details.
  4. Click .
  5. Then set up the report rows and columns. 
    • The rows control the structure of the report. There are different row types, such as Account row, Total row, Text row.  Account rows (and rows) specify the selection of General Ledger accounts (or a selection of ) to report on.  Other row types determine where other lines such as subtotals and text lines come.
    • The columns control what information is shown on each line.  You can select from a number of different data items, and specify which row types each one is shown on.
