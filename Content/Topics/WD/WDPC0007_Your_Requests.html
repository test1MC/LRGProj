Your Requests

The Previous Requests screen shows details of requests you have made.  This includes the status of each request.  You can also create a new request by copying the details from a previous request.

To see your requests:

  1. Click .

    displays the Previous Requests screen, which lists any requests you have made in the last 90 days.

    Name shows the site name.

    Status shows the status of the request.  This could be:

    ApprovedThe request has been accepted and is being dealt with by the . 
    Awaiting ApprovalAn approval workflow is being used and the request has not yet been approved.
    Approval RejectedAn approval workflow is being used and the request has been rejected.
    Returned from The entire request has been passed back from the .  You can copy the details to a new request as explained below.
  2. To see the details of a request, click the link in the Request Number column.

  3. To create a new request, using the same products as the request you are looking at, click Copy to basket.

    This adds the items from the request to your basket, with a default Required From date of tomorrow, and a default Expected Date of one week from tomorrow.  You can change or delete these request lines (see Your Basket).
