Defining a Standard Subcontractor Certificate

To define a standard subcontractor certificate:

  1. Go to Standard Certificate Definition.
  2. Click to enter a new standard certificate.
  3. In the Type field, enter a code to identify the certificate type.

     

    One common convention is to use A as your main Standard Definition, and B, C, as variations on this, and to use M for Matrix-type subcontracts used in the VAP system.
  4. Enter a brief description of the certificate type.
  5. Leave the other fields blank and click on the Detail tab.
  6. displays the line definition screen. Click to create a new line and enter the Line details as required. Click when the line is complete.
  7. When the definition is complete return to the Main tab and enter the required line numbers.
