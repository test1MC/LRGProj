Positive Pay – Configuration

Parameter

Description

CB/

Use Positive Pay Y/N

Will positive pay enhancements be used? If this is not set to Yes, the old-style Positive Pay Report will be used.

CB/POSEMPRP

Positive pay detail report to show Employee Name Y/N

If both the CB parameters USRPOSYPY and POSEMPRP are set to Yes, show the payroll Employee name on the report otherwise just show the word EMPLOYEE.
