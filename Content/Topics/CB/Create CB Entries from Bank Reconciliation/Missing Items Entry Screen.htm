Missing Items Entry Screen

A new The Enter Missing Items screen allows you to enter any missing items to the value of the bank statement item.

Enter Missing Items Screen

The context area shows the batch number of the open batch.

The record header section shows the value of the statement item, the value of the missing items entered so far, and the difference between the two, which informs you what value is still to be entered.

The browse allows you to enter details of the missing items. It shows only the items that are linked to the selected statement item (by the shared internal reference). Other items that may exist for different statement items are not shown (but can be viewed in Transaction Entry or ).

Enter Missing Items - Detail

Some fields are pre-assigned:

  • The number (if there is one) and date are taken from the statement item.
  • The deposit date is set to today’s date.
  • Whether the item is a receipt or payment is dependent on the signage of the statement item.
  • The description automatically defaults to the statement line description when CB/STTYDESC = Y.

Apart from the number (if there is one) and date, all other fields are updateable.

You can add, maintain and delete item records until you are happy that all missing items have been entered. Entering a missing item is exactly the same as entering a normal transaction - you can enter header and analysis details and also allocations if desired.
