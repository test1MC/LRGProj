8 Exercise 8 - Adding Gauges to Pages

Colour ranges are set up as global settings which are company specific. For this reason we will not have to set them up on the training environment. However for this exercise to work when you load them into your local environment you will have to set up the ranges in your own environments.

 

8.1 Check ranges exist.

OA Reporting & BI Setup menu –

  • Company Specific
  • Data Display Colour Ranges

 

For Gauges, is a hexadecimal RGB colour Code.  A full list of RGB codes is available in the General Documentation Guide.

 

Add gauge to Cost field.

Amend the Costs TD field on the page on Exercise 1.  Remove the Class entry and in the populate field enter the gauges range to be used to use the new colour range and set the View-As to Gauge-Circular Coloured
