12.4 Add the jc_job Table as a header table.

 

Header tables should include any table used in the calling procedure.  The header tables record for your in-line report should now read

Header Tables jc_config,co_config,jc_job

QueryFOR EACH jc_costcode WHERE jc_costcode.kco= {kco}

AND jc_costcode.job_num = '{job_num}'
