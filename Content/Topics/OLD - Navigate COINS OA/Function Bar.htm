Function Bar

If you minime a frame, it moves to the "Function Bar", an area at the bottom of the screen.

The Function Bar hides automatically, but you can point to the status bar at the bottom of the screen to reveal the Function Bar with the minimed functions on.

Click the restore or maximize button on a frame to expand it again.

 

Function Bar

Although the only way to run functions in frames is from a Desktop tile, if tiles have been minimised, the Function Bar is available from standard screens, and functions in frames can be restored on top of standard screens.
