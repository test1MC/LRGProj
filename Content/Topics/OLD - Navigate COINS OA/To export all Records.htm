To export all Records

  1. Locate the Browse screen you require.  Make sure no filters are applied
  2. Select the Export button at the bottom of the screen. A message will be displayed asking “Do you want to open or save this file?”.

  3. Select the button you require. The Open button will display the information on the screen, alternatively the Save button will enable you to save the data file on your network drive.
  4. In this example we will select the Open button.

  5. The data can be filtered, amended or changed as required.

Note

The data is not linked to the Database as the records have been exported outside .
