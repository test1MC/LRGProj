Urgent


The Urgent output type enables you to run the report quickly. There are two types of queues, Urgent and Normal. By selecting the Urgent button, the report will jump straight to the Urgent queue be processed in the background and store the report in the Report Status workbench.

You should be selective and use the Urgent button sparingly, as the majority of reports can be run using the other options. This option is usually restricted to certain users.
