Delayed


The Delayed output type enables you to run the report at a specified time, usually quiet times, i.e. Lunchtimes or Evenings. It will run the report in the Background and store the report in the Report Status workbench.

  1. Selecting the Delayed radio button will display a new set of fields.
  2. Enter the date and time you wish the report to be collated. The data will be accurate at the time the report is run, not the time you submit the request.
  3. When you select the Next button a notification message will confirm you selected a Delayed Report and the time the report will run. 
  4. A copy of the report will be available in the Report Status workbench for you to review.

    5. The status will be Waiting until the allocated time and will change to Complete.
