Global Search

NOTE

The Search frame is only available if you have a Desktop configured and if global search is enabled by the configuration of both your system and your user record. The results returned by the search are also subject to your systems configuration.

You can launch the Search frame by:

  • Pressing Ctrl+Shift+?.
  • Clicking the button on the Desktopin the page header. 

You can type a word or words to search for.  As you type, displays matching records, grouped by index type.

If you cursor up or down, or move the mouse pointer over a record then a detail frame to the right shows more detail about that record.

You can filter the search to show only records that belong to the categories you select.

remembers your filter settings and the text you searched for, and uses them the next time you open Global Search (until you log out).

The detail frame is configurable and can show different information depending on the type of record.

Clicking the link on the item or pressing Return on the highlighted item launches the corresponding link page.  This can open in in a new frame, in a new tab, or in the current main area. 

Alternatively you can right click to choose to open it in a new tab or new window.

The results will match words that begin with your search words.  You can type the words you want to search for in any order.  For example:

report balance or balance report or bal rep
manchester const or manch construction

To search for an exact match, enclose it in quotation marks.  For example, to search for names that contain the whole word "sub", but not "subcontract", type "sub".

You can also use | (pipe) to represent "or", and & to represent "and". For example:

(John|Sally) Smith       will search for John Smith and Sally Smith.

You can enter the module code to help identify functions with similar names (for example, posting report CS will show only posting reports).

If the total number of matches is less than 500, the results show all matches.  If the total number of matches is 500 or more, only the first 10 results from each category are shown, and if there are more than 10 results in a category, a more... link takes you to a full-screen browse of those records.

Global Search and Function Search are also available as “traditional” OA functions, on the System > System Setup menu.
