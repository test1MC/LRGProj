Scheduler Tab

The Scheduler tab enables you to view all reports that have been scheduled to run, i.e. every month end. These frequencies will have been predefined and only available for selection. 

The list will show the date range when the report is required. This is not the frequency but the live date range, i.e. every month from 1 January to 1 July.

You can also run these reports on an ad-hoc basis by selecting the Run Now button. All the filters will apply.
