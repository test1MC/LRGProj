Deleting a Message

Whilst it is recommended that any messages added should have an expiry date, there will be some circumstances where you have to manually delete the message.

  1. Click in the Message Board area of the home page
  2. The Message Board screen is displayed, listing all the messages created, regardless of the user.

  3. Locate the message required. Select the Tick Box adjacent to the message. Select the Delete button.
  4. The system will display the warning message

  5. Select the OK button. The screen will automatically refresh and the message will be removed.

You will need to refresh the Home Page for the message to be removed from there too.
