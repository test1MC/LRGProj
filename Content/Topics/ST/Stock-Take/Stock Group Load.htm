Stock Group Load

To get the file template for Stock Groups navigate to Stock > Stock Groups and click "Bulk" + "Add"

 

 

Then click "Save" towards the bottom of the screen

 

 

Then click "Open" towards the bottom of the screen

 

 

This opens up a .xml file in Microsoft Excel that is available to be edited (once you've click the "Enable Editing" button)

 

 

Edit the file as required, with the data for the Stock Groups

 

 

Copy all of the data and paste it into the box you see in when you click "Bulk" and "Add". Thereafter, click "Save"

 

 

The Stock Group has now been loaded into
