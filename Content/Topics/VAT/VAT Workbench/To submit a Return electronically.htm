To submit a Return electronically:

  1. Go to VAT Returns Workbench.
  1. On the VAT Returns tab, select the Return you want to submit.

  1. Choose the Submit VAT Return Electronically action.  

This shows the figures that will be returned.

  1. Click save

Any adjustments to the Return or other input on this screen will be saved.

 

 

This screen can be backed out of the adjustments here are not posted with the GL posting

On Save submits the Return electronically to the Government Gateway

 

Once a VAT Return has been submitted to HMRC, it cannot be cancelled within (except as noted under Cancelling a VAT Return).  Note that there is no facility for to send an amendment or cancellation of a submitted VAT Return to HMRC.  If a VAT Return which has been submitted to HMRC needs to be amended or cancelled, you will need to contact HMRC.
