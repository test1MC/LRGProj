SY/FTPUNAME - Generalised FTP user name

(plus only) The user name of an account that allows access to the host computer using FTP.

Print Previewer and a number of programs (such as syget, syput and transfer) require FTP access to the host computer. If you have a security policy that prevents regular users accessing the host computer through FTP, set up a special FTP account (typically with no telnet login) and enter the username and password here. The programs that need FTP access will know about the account through these parameters, but it will be hidden from normal users.
