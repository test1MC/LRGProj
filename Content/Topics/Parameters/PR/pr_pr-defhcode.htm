PR/PR-DEFHCODE - Payroll Default Hold code

If this parameter has a value, the invoices created as part of payroll posting will be put with the "hold code" assigned here. The hold code must be a valid commercial hold code entered in .

For US only: For more information on this parameter and auto invoicing, see the PDF How To Guide entitled Use Auto Invoicing in OA Payroll (US Only) found in Payroll How To Guides.
