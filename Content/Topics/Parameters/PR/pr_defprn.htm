PR/DEFPRN - Use Printing Preferences for Printing PR Payslips [ALL]

Whether to use printing preferences when printing PR payslips (Y/N), such as allowing PR payslips to be emailed to employees.

For detailed information on how to be able to email secure PR payslips to employees, see the PDF How To Guide entitled Email Secure PR Payslips to Employees in OA found in Payroll How To Guides.
