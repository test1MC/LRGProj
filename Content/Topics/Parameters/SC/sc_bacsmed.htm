SC/BACSMED - BACS Media (1 4 5 or 6)

The medium for files.

The options are:

BACSMED can have a second entry, which is text that will be put in to the application field (a field that appears in the header identifying the application used to generate the file). By default puts the module code. If the CO parameter BACSAPPL is set to Y, appends the number in the application field.
