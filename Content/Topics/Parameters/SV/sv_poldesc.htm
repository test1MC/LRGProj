SV/POLDESC - Description format to be used for SC Amendment Variation Order

This determines the description of each line on a subcontract amendment order created from a subcontract .

All fields from the Item and Resource record can be used be enclosing them in curly braces. A pipe followed by a number can be used to specify the number of characters to show. For example:

{svi_desc} {svr_desc|15}

would show all of the Item Description (svi_desc) and the first 15 characters of the Resource Description (svr_desc).
If this parameter is blank the following format is used:

{svi_desc} - {svr_desc}
