CO/BACSPAYC - Create a BACS Pay Code for each BACS file

Whether to create separate pay codes for files.

This is to cater for multiple payments on the same day and is currently only relevant for one Middle-East bank.

If BACSPAYC is set to Y:

  • The first PL payment batch sent through against a particular bank will have Payment Set Code = P01. The second will have P02; the third, P03 and the fourth will be P01. It will then continue to loop between these codes.
  • The first SC payment batch sent through against a particular bank will have Payment Set Code = S01. The second will have S02; the third, S03 and the fourth will be S01. It will then continue to loop between these codes.
