HS/TELORD - Browse telephone order sequence in OA

( only) The order of priority for displaying a person's telephone number.

If there is only one column for displaying a telephone number, this determines the order in which looks for a number.

For example, if the parameter is set to M,W,H, displays the mobile number if there is one; if there is no mobile number, displays a work number, and if there is no work number, displays the home number.
