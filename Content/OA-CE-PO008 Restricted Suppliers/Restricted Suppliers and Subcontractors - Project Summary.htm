Project Summary

If either of the PI parameters SUPINUSE or SUBINUSE is set to Y, a new tab entitled “Restricted Lists” appears in the “Details” tab group, immediately to the right of the “Procurement Strategy” tab. (The "Restricted LIsts" tab is only shown if the project is of type “”. In other words, it is not shown if the project is a “lead” or a “”.)

On selecting the “Restricted Lists” tab a sub-tab strip appears below. This sub-tab strip contains a tab entitled “Suppliers” (if PI/SUPINUSE is set to Y) and a tab entitled “Subcontractors” (if PI/SUBINUSE is set to Y).
