Sectors Set up

The VAT workbench requires all GL sectors to be set up relevant sector details.

GL Elements:

GL Elements allows you to maintain the codes that are valid for each of the elements of General Ledger account codes

Sector Detail Maintenance

Sector Details Maintenance allows you to set up the addresses and other details for each sector including Vat.

Associated Parameters

GL/SECTOR

GL/SECTOR (Current Value: Y)

Set this to Y if you wish to use sectors.

See also the parameters SECTGLEL, SECTICO and SECTTYPE

 

Sector Maintenance Set Up

Click on the Look up […] this open a window to the VAT registrations.

Select the Appropriate VAT registration for the sector and save. If a VAT registration is not available this will need to be added to VAT registrations

Important – Every GL Element requires a Sector
