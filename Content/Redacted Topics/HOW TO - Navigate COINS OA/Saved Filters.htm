Saved Filters

From v11.06 A new button is now available in the footer button bar.

There are no parameters to enable this feature – it is automatically made available generically on all browse screens that have filtering.

 

If you press this new button then a dialog will allow you to save the current active filter. You should enter and apply the filter before pressing the save filter button.

Select Create New Filter and enter the filter name. On OK the screen is refreshed and the filter is selected.

The additional delete button alongside allows you to delete the saved filter.

If you press the save filter button again then you get to select to overwrite the existing filter with a new filter or to choose “Create New Filter” and type in a new one.

Advanced filters can also be saved.

You are then able to select a saved filter in the filter button bar.

When you select a filter it is the same as filling in the filter fields and applying the filter button.

The following filter elements are saved:

• Simple/Advanced Type

• Named Filter(s)

• Filter Field

• Filter Value

• Advanced Filters

• Sort Order

• Body Selectors (if used)
