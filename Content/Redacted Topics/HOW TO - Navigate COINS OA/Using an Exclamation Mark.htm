Using an ! Exclamation Mark to Filter

Exclamation Marks are used to exclude records, words, codes or values from a list. These are looking for a match in the Search field when excluding data.

The ! Filter

Using this wildcard would normally be used in conjunction with others, as if you are excluding something, you would require everything else, for example:: exclude all orders on the 14th April 2010 however show me everything else.

When using the exclude Filter this has to go before any other search filters as shown above.
