My Files Tab

My Files is a list of files in your 'home' area on the server. Some functions, such as export routines, will place files in this area.

 

When you first visit this page, creates your home area: a directory on the server, under the $BASE directory, called /var/home/<your-userID>.

Field

Description

File

The name of the file in your home directory.

Click the file name to download the file to your device

Size

The size of the file

Modified

The date and time that this file was created/modified
