Account Name and Image

The header of the desktopbar shows your user name or user ID (depending on the setting of the SY parameter SHOWUSER), and an image to represent your account.

Account Name and Image

You can change the account image by clicking it to bring up a box and browsing for a new image:

Upload Picture Dialogue Box

You can change your account image and display name by selecting Change Proifle from the user menu.

This brings up a box where you can browse for a new image and enter a different display name.

The image must be a JPG file, and ideally should be approximately 60px by 60px. You can use larger images but the browser will resize them, which does not always produce good results.

The account image can also be configured by the system administrator; see 3.2.3, Images. You can click the user name (or ID) to open the User Preferences screen.
