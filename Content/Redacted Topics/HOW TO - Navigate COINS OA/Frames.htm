Frames

Some functions, when run from the Desktop, open in a frame, which you can resize, maxime or minime.

Function in a Frame

The of the frame's title bar matches the of the tile from which it was run.

Buttons are available in the title bar to close, maxime, minime or refresh the frame, and to show the side-frame help for the function.

Frame Buttons

You can resize a frame by dragging the handle in the bottom-right corner:. remembers the size of the frame; the next time you run the frame, it opens with the new size.
