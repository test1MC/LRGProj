Images

You may want to include images, such as a company logo, on the document.

To include an image called filename.gif:

In the Field field, enter:

IMAGE^filename (if the image file exists in the /images/ directory of the installation).

IMAGE^/relative-path/filename (if the image exists in a different directory on the webserver; relative-path is the relative path on the webserver to the directory where the image is stored).

The image files must be gif images, and the file names of the image files must be in lower-case. Do not add the .gif extension after IMAGE^filename

 

Specify the Height and Width of the image.

Select View As: PICTURE.

If you want an image that relates to the whole page of a document (for example: a watermark, or a logo that shows down the side of the page), set this up on the Report Page Layout used for the document.
