Colours

In HTML and FOP you can use either names or hexadecimal codes to identify .

Colour names are simply the usual names for , such as red, green, silver, brown. Other colour names include goldenrod and fuchsia. (There are several websites that list the colour names you can use; search for "HTML colour names". Some browsers may not support all the names.)

Hexadecimal codes are written using the format #aabbcc, where aa, bb and cc are hexadecimal (base 16; 0 to 255) values that represent the amount of red, green and blue in the . So for example:

  • #00FF00 is pure green (0 red, 255 green, 0 blue)
  • #808080 is a mid grey (128 red, 128 green, 128 blue)
  • #310073 is a dark purple (49 red, 0 green, 115 blue)

This allows you to specify much more precisely than using the colour names. Many graphics programs allow you to select a colour from a palette, and will tell you what the hexadecimal code for that colour is.

The used on the standard OA screens (pre-v1105) are:

  • The predominant light blue is "#DCE4EE"
  • The mid blue (used for example on column headers and the selection areas at the left of a browse) is "#6699CC"
  • The dark blue (used in the title bar) is "#3A3F8C".
  • White is "#FFFFFF" and black is "#000000".
