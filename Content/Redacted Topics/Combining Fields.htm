Combining Fields

You can append one data field to another, using a calculation. For example, you might want to print all the address fields on a single line, or print the contract number and contract name as if they were a single field.

To enter a calculation:

  • Add a new line.
  • Leave the Field field blank.
  • Enter the calculation in the Calculation field. Put a $ before each part of a calculation that involves text strings; use the full table and fieldname.

For example: $a=jc_job.job_num; $b=" (" + jc_job.job_name + ")"; $a + b
will display the contract number, followed by the contract name in brackets.
