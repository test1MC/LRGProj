Importing and Exporting Documents

provides procedures for importing and exporting document definitions (for example, to allow you to transfer definitions from a test environment to a live environment).

To export document definitions:


Select System, Report Setup, Export Document Definitions.

Specify the document definitions you want to export by using the lookup, or key the name in directly if known.

Select the Forward button

displays the document definition data in the Document Data field.

Select all the text in the field (click in the field and press Ctrl+A).

Copy the selected text and paste it into a file or an email message.

You can then transfer this data to a different environment.

To import document definitions:

Select System, Report Setup, Import Document Definitions.

Paste the text in the Report Data field

and select Save

The imported document definitions will now be available in Document Designer.
